# rafolio
Raf Tan Portfolio

# Environment Setup
* node
* forever
* bower
* gulp

# NPM Instructions
* `cd rafolio` to Start
* `sudo npm run dev` for development
* `PORT=8080 npm run start` for server dev
* `sudo npm run start` for server prod


# Gulp
* `gulp` for development
* `gulp prod --env=prod` for production
